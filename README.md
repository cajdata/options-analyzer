# My Options App

A minimal FastAPI + OCR + Web Frontend example to parse and analyze options data from screenshots.

## Prerequisites
- Python 3.8+ 
- [Tesseract OCR installed](https://github.com/UB-Mannheim/tesseract/wiki)
- pip install -r requirements.txt
